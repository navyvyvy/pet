# Design

## Theme

Restrained product UI for a small desktop utility. The physical scene is a user adjusting a quiet tool beside active work, in normal room lighting, without wanting a full-screen product moment.

## Color

Use OKLCH tokens. The interface uses pure white as the main background, a soft cobalt primary for selection and primary actions, green only as a secondary accent, and orange/red only for warnings or destructive actions.

## Typography

Use one system sans stack for all labels, fields, buttons, and headings. Keep headings compact and fixed-size.

## Layout

Settings is a task surface: toolbar, app settings band, character list or editor, then preview grid. Cards are only used for repeated pet rows, empty state, and frame tiles.

## Components

Buttons use a shared 6px radius and consistent icon placement. Inputs and selects use the same height, border, and focus ring. The pet bar surface is transparent and unframed.

## Motion

Settings UI avoids decorative motion. Pet movement is the core behavior and can be paused. Reduced-motion users should not receive extra UI animation.
